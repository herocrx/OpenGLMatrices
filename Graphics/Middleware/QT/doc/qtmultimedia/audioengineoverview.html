Positional Audio

QtAudioEngine Features

Qt Multimedia includes the QtAudioEngine QML module for providing 3D positional audio playback and content management.

QtAudioEngine enables developers to organize wave files into discrete Sound with different play variations, group sound controls by AudioCategory categories and define attenuation models and various 3d audio settings all in one place. Playback of sound instances can be conveniently activated by in-app events and managed by QtAudioEngine or controlled by explicitly defining SoundInstance for easier QML bindings.

To access these QML types import the QtAudioEngine 1.0 module.

import QtQuick 2.0
import QtAudioEngine 1.0

AudioEngine {
//...
}
